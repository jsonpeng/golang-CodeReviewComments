##Gofmt
  你可以在你的代码中运行 [Gofmt]("https://golang.org/cmd/gofmt/" "悬停显示") 以自动修复大多数代码格式问题。几乎所有的代码都使用 `gofmt`。<br>
  另一种方法是使用[goimports]("https://godoc.org/golang.org/x/tools/cmd/goimports" "悬停显示")，它是 `gofmt`的超集，可根据需要添加（删除）行
